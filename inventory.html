Quick Start Guide

  • Import your JSON to load current inventory.
  • Quick stock updates — Use + and - buttons on any item (stock > 0 only for menu copy).
  • Full edit — Tap "Edit" for name, description, weight, origin, rarity, clearance toggle (with date), etc.
  • Performance flags — Green (sold ≤7 days), Yellow (7-29 days), Red (30+ days or never sold).
  • Copy functions — "Copy Menu Data" and "Copy SMS Menu" exclude out-of-stock items.
  • Export JSON — Do this after every session to save changes.
