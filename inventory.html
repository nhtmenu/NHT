Full User Guide

Welcome to Never Had That Inventory Manager

  • + Add New Item — Adds a blank item at the very top of the list (no category by default).
  • Edit Item — Tap "Edit". The item is replaced by the full edit card **in its exact position**. Other Edit buttons are hidden while editing.
  • Stock Changes & Sales Logging — Lower the Stock number and hit "Save & Close" to auto-log the sale. Stock color changes: ≤10 yellow, ≤5 orange, 0 red.
  • Profit Since Last Update — Shows only sales from this session. Resets automatically on import.
  • All-Time Sales History — Persistent history with graph (filter by time period) and full log. Saved in JSON.
  • Category Headers — Tap to collapse/expand. Uncategorized items stay at the top.
  • Export JSON — Always export after changes to save everything.
  • Import JSON — Loads data. Current session profit starts at $0.
  • Copy Menu Data / Copy SMS Menu — Ready for website and texting.

Best Practices: Edit at end of day, export regularly, use graph to spot trends.
